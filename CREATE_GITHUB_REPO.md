# Create GitHub Repository - Step by Step

## Follow these exact steps:

---

## Step 1: Open GitHub

1. Open your web browser (Chrome, Firefox, Edge, etc.)
2. Go to: **https://github.com/new**
3. If prompted, log in with your GitHub account (username: ManuelRodrigues0)

---

## Step 2: Fill in Repository Details

You will see a form with several fields. Fill them in exactly as shown:

### Required Fields:

**Repository name:**
```
skillhive
```

**Description (optional but recommended):**
```
SkillHive - Credit-based skill exchange platform
```

**Visibility:**
- Select **Public** (this makes it free and accessible to everyone)

### ⚠️ IMPORTANT - Do NOT check these boxes:
- ❌ **DO NOT** check "Add a README file"
- ❌ **DO NOT** check "Add .gitignore"
- ❌ **DO NOT** check "Choose a license"

Leave all three unchecked!

---

## Step 3: Create Repository

1. Scroll down if needed
2. Click the green button: **"Create repository"**

---

## Step 4: Verify Creation

After clicking "Create repository", you should see:
- A page with the title "skillhive"
- Text saying "Quick setup — if you've done this before"
- Instructions for pushing code

**The page will show something like:**
```
…or push an existing repository from the command line
git remote add origin https://github.com/ManuelRodrigues0/skillhive.git
git branch -M main
git push -u origin main
```

## ❓ Common Issues

**Can't see the "Create repository" button?**
- Make sure you're logged into GitHub
- Try refreshing the page

**Repository name already taken?**
- Try: `skillhive-platform` or `skillhive-app`

**Don't have a GitHub account?**
- Click "Sign up" at the top right of github.com
- Follow the registration process (it's free)
